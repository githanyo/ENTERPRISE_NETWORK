# Networking Labs

This repository contains Cisco Packet Tracer networking labs created to strengthen practical understanding of networking concepts.

## Labs Included

### EtherChannel Lab
Objective:
- Configure EtherChannel using LACP
- Understand link aggregation
- Test redundancy and load balancing

Skills practiced:
- Switch configuration
- LACP
- Trunking
- Interface verification

---

### HSRP Lab
Objective:
- Configure HSRP between routers
- Understand gateway redundancy
- Test failover mechanisms

Skills practiced:
- Router configuration
- HSRP
- Gateway redundancy
- Fault tolerance

---

## Tools Used

- Cisco Packet Tracer
- CLI configuration
- Basic troubleshooting commands

## Verification Commands

Common commands used:

```bash
show etherchannel summary
show standby
show running-config
ping
```
---
## Learning Outcome

These labs demonstrate two important high-availability concepts:

- EtherChannel → Link redundancy and bandwidth aggregation
- HSRP → Gateway redundancy and automatic failover
